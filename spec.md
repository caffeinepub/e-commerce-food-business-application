# E commerce Food Business Application

## Overview
A web application for selling packaged dry fruits, snacks, and chocolates with comprehensive admin catalog management and customer ordering capabilities. The application content is in English.

## Product Categories
- Dry Fruits
- Snacks  
- Chocolates

## Admin Features

### Admin Dashboard
A dedicated admin interface for complete catalog and menu management with full user-facing functionality.

### Product Catalog Management
- Add new products with name, price, description, image upload, stock status, and category selection
- Edit existing product details including all product attributes
- Delete products from catalog with confirmation
- Organize products by the three main categories
- Real-time updates to the storefront when products are modified
- Stock status management (in stock, out of stock, low stock)

### Order Management
- View all customer orders
- Update order statuses (pending, processing, shipped, delivered)
- Track regular and bulk orders separately

## Customer Features

### Home Page Product Display
- Products displayed under category-based tabs (Dry Fruits, Snacks, Chocolates)
- Automatic updates when admin modifies products
- Category-based navigation and filtering

### Product Browsing
- Browse products by category tabs on home page
- View detailed product information including images, descriptions, prices, and stock status
- Search and filter products
- Stock availability indicators

### Shopping Cart & Orders
- Add products to cart (with stock validation)
- Select order type: regular (pre-prepared) or bulk (catering-style)
- Place orders with customer contact information
- View order confirmation

## Backend Data Storage
- Product catalog with name, price, description, image references, stock status, and category
- Product categories (Dry Fruits, Snacks, Chocolates)
- Customer orders with items, quantities, order type, and status
- Order status tracking
- Stock level management

## Backend Operations
- CRUD operations for comprehensive product management
- Image upload and storage for products
- Stock status updates and validation
- Order creation and status updates
- Product retrieval by category for home page tabs
- Order history and tracking
- Real-time product updates to storefront
